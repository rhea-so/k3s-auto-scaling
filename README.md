## Configuration & Setup

### Project Cloning

```sh
git clone https://github.com/rhea-so/K3sAutoScaling
cd K3sAutoScaling
npm i
```

## Build 

```sh
tsc
```

## Test

```sh
npm test
```

### Get Test Coverage

```sh
npm run coverage
```

### Artillery

```sh
artillery quick --duration 60 --rate 10 -n 20 http://192.168.0.36:3000/test.html
```
